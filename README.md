# VideoSnap

VideoSnap is a Next.js 15 App Router scaffold for a premium video-downloader SaaS.

## Stack

- Next.js 15 + TypeScript
- Tailwind CSS v4
- Supabase schema and route scaffolding
- Vercel deployment config

## Project Layout

The app lives in the `videosnap/` folder inside this workspace.

## Environment

Copy `.env.local.example` to `.env.local` and fill in the values before deploying.

## Notes

- The scaffold includes marketing pages, a download flow, admin routes, API endpoints, and SEO/security files.
- On this machine, Windows shell tooling is incomplete because `C:\Windows\System32\cmd.exe` is unavailable, so dependency installation and build validation are blocked here.
